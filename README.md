# Etch-A-Sketch

A browser-based drawing grid built with vanilla HTML, CSS, and JavaScript, inspired by the classic Etch-A-Sketch toy. Built as part of The Odin Project curriculum.

## Features
- Adjustable grid size (1–100 squares per side)
- Draw by hovering over squares
- Eraser mode to undo drawing
- Custom color picker
- Random single-color mode
- Rainbow mode (random color on every hover)
- Reset button to clear the whole grid instantly

## How to Use
1. Enter a grid size (1–100) in the input box.
2. Click "Create Grid" to build the grid — it only updates on click, not as you type.
3. Hover your mouse over the squares to draw.
4. Use the toolbar buttons to switch between Pen, Eraser, Color Picker, Random, and Rainbow modes.
5. Click "Reset" anytime to clear the grid back to white.
